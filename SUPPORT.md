# Support

If you need help using Cortex:

1. Check [README.md](README.md) for install and quick-start.
2. Check [CONNECTING.md](CONNECTING.md) for integration setup.
3. Check [Info/team-mode-setup.md](Info/team-mode-setup.md) for team deployments.

If you found a bug:
- Open a GitHub issue with reproduction steps, OS, and expected vs actual behavior.
- For security reports, use [SECURITY.md](SECURITY.md) (private disclosure preferred).

If you want to contribute:
- Start with [CONTRIBUTING.md](CONTRIBUTING.md).
- Pick a task from [Info/roadmap.md](Info/roadmap.md).
